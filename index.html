📖 使用帮助

从零到一，快速搭建达人信息管理系统

📋 第一步：创建 Google Form

  1. 1访问 forms.google.com，点击 "+" 新建表单
  2. 2设置标题："达人信息收集系统"
  3. 3添加字段：分类（下拉）、地区（单选）、达人账号、主页链接、提报人、备注
  4. 4确保"地区"字段选项与工作表名称完全匹配
  5. 5点击"发送"获取表单链接或嵌入代码

📊 第二步：连接 Google Sheets

  1. 1在 Form 编辑页，点击"Responses"（回复）标签
  2. 2点击绿色表格图标"Link to Sheets"
  3. 3选择"Create a new spreadsheet"
  4. 4系统自动创建含"Form responses 1"工作表的 Sheets
  5. 5复制 Sheets 链接并粘贴到系统配置页

🔄 第三步：设置 QUERY 分流公式

  1. 1为每个地区点击底部"+"新建工作表
  2. 2将工作表重命名为地区名（如"菲律宾"）
  3. 3在 A1 单元格输入 QUERY 公式
  4. 4共需创建约 17 个地区工作表 + 1 个总提报表
  5. 5每次表单提交，所有工作表自动更新

🚀 第四步：部署到 GitHub Pages

  1. 1访问 github.com/sdyy511/influencer-manager
  2. 2点击"Add file" → "Create new file"
  3. 3文件名填 index.html，粘贴本系统代码
  4. 4提交后等待约1分钟，GitHub Pages 自动生效
  5. 5访问：sdyy511.github.io/influencer-manager/

🔧 常见问题排查

  1. Q公式不工作？检查工作表名称是否与地区字段值完全一致（区分全/半角）
  2. Q数据未出现？确认 QUERY 的列标（B列）对应"地区"字段
  3. QForm 无法嵌入？检查 Form 权限是否设置为"任何人可提交"
  4. Q性能缓慢？将 QUERY 范围从 A:Z 缩小为 A:G

💡 QUERY 公式速查

-- 全部数据（提报表） =QUERY('Form responses 1'!A:G, "SELECT * WHERE 1=1", 1) -- 单地区筛选 =QUERY('Form responses 1'!A:G, "SELECT * WHERE B = '菲律宾'", 1) -- 多条件筛选 =QUERY('Form responses 1'!A:G, "SELECT * WHERE B='墨西哥' AND C='生活类'", 1) -- 各地区数量统计 =QUERY('Form responses 1'!A:G, "SELECT B, COUNT(B) GROUP BY B LABEL COUNT(B) '数量'", 1)
